VERA simulation of the MIT BEAVRS measured full-core benchmark (cycles 1-2):
zero-power physics tests, critical boron, and flux maps vs measured plant data.
Co-authored by Palmtag. Like the WBN1 reports, this is measured full-core
neutronics — valuable context and a model of real validation, but outside the
problem class netflow's current physics can reproduce.
